# Final Project - Python Analysis

## Описание
Анализ данных с использованием Python и библиотек pandas, numpy, 
matplotlib, seaborn, scikit-learn.

## Данные
- `data.csv` — основной датасет для анализа

## Задания
1. Первичный анализ данных (EDA)
   - изучение структуры данных
   - описательная статистика
   - анализ пропущенных значений

2. Визуализация данных
   - распределение переменных
   - корреляционный анализ
   - графики зависимостей

3. Предобработка данных
   - очистка данных
   - обработка выбросов
   - кодирование категориальных переменных

4. Машинное обучение
   - построение модели
   - оценка качества (MSE, R², Accuracy)
   - интерпретация результатов

## Инструменты
- Python 3.x
- Jupyter Notebook
- pandas, numpy
- matplotlib, seaborn
- scikit-learn

## Структура файлов
- `analysis.ipynb` — основной ноутбук с анализом
